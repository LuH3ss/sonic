# Project Specification: Sonic-SaaS 2026

## Core Vision
Un dashboard de análisis musical "AI-First" con estética Industrial Dark. 
El objetivo es permitir a los artistas visualizar sus datos de streaming 
con una experiencia de usuario (UX) de nivel Apple/Linear.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Framer Motion
- **Components:** Shadcn UI (Customized)
- **Database:** Neon (PostgreSQL) + Prisma ORM
- **AI:** Vercel AI SDK (Claude 3.5 Sonnet integration)
- **Icons:** Lucide React

## Visual Identity (The "Vibe")
- **Background:** #030303 (Onyx)
- **Cards:** #080808 con bordes de 1px solid #1a1a1a
- **Accent:** #6366f1 (Indigo Electric)
- **Typography:** Geist Sans & Cal Sans
- **Transitions:** 0.4s ease-out con resortes (springs)
